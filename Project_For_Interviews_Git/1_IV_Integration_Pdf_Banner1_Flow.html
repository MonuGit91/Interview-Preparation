⚠️ Unified Exception Translation & Fault Tolerance

1. Downstream Interception
All wrappers monitor HTTP statuses from Directory Services and Content Server. If a non-2xx code returns, it is caught immediately and translated into an ExternalApiException.
2. Centralized Advice
A unified controller advice class intercepts ExternalApiException, hides the internal java stack traces, maps the error to a clean JSON and sends the exact matching HTTP code to clients.
3. Resilience Policies
Using Resilience4j, client integrations implement Circuit Breakers and Retry with Exponential Backoff, isolating thread resources during outages.
